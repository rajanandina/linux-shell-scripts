# Linux System Monitoring Script

This project is a Bash script that monitors system health.

## Features
- Shows current date and time
- Displays CPU load
- Shows memory usage
- Shows disk usage
- Displays system IP address

## Technologies Used
- Bash Scripting
- Linux commands (top, free, df, hostname)

## How to Run

chmod +x system_monitor.sh
./system_monitor.sh
